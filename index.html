서버 소개

공블몬 서버는 포켓몬의 세계관과 크리에이트 모드의 자동화 기능을 결합한 독특한 게임 경험을 제공합니다.

다양한 최적화 모드를 통해 부드러운 게임플레이를 즐기면서, 포켓몬을 수집하고 훈련시키고, 복잡한 자동화 기계를 설계하세요!

Cobblemon과 Create 모드의 조화로운 융합을 통해 창의력을 발휘하고 새로운 차원의 마인크래프트 경험을 누려보세요.

수많은 모드가 완벽하게 최적화되어 있어 원활한 게임 플레이를 보장합니다. 포켓몬 배틀과 자동화 시스템을 동시에 즐기면서 자신만의 스토리를 만들어보세요.

주요 특징

공블몬 서버가 제공하는 특별한 경험을 알아보세요.

Cobblemon 세계

야생에서 다양한 포켓몬을 발견하고 수집하세요. 트레이너와 대결하고, 체육관 챌린지에 도전하며, 자신만의 포켓몬 마스터가 되어보세요!

시스템 요구사항

최소 사양

  • CPU: Intel Core i3-3210 / AMD A8-7600 이상
  • RAM: 8GB (최소 4GB 할당)
  • 그래픽: 내장 그래픽 (최소 1GB VRAM)
  • 저장공간: 5GB 여유 공간
  • 인터넷: 안정적인 브로드밴드 연결

권장 사양

  • CPU: Intel Core i5-6600K / AMD Ryzen 5 1600 이상
  • RAM: 16GB (최소 6GB 할당)
  • 그래픽: NVIDIA GeForce GTX 1050 / AMD Radeon RX 560 이상
  • 저장공간: SSD 10GB 여유 공간
  • 인터넷: 10Mbps 이상 안정적인 연결

Java 요구사항

  • Java 버전: Java 17 이상
  • JVM 인수: 권장 JVM 인수 사용 (모드팩에 포함)
  • 모드로더: Fabric 0.15.0 이상

시작하기

공블몬에 접속하는 방법을 알아보세요. 몇 가지 간단한 단계를 따라 빠르게 시작할 수 있습니다.

1

모드팩 다운로드

위의 '모드팩 다운로드' 섹션에서 공블몬 모드팩을 다운로드하세요. 이미 필요한 모든 모드가 설치되어 있습니다.

2

모드팩 설치

모드팩 프로그램을 켠 후, 모드팩 설치 버튼을 누르세요. 나머지는 모드팩 설치 프로그램이 모두 해줄거에요! (오류시 디스코드로 문의 부탁드립니다)

3

Java 설정 확인

마인크래프트가 Java 17 이상을 사용하도록 설정되어 있는지 확인하세요. 또한 RAM 할당을 최소 6GB 이상으로 설정하는 것이 권장됩니다.

4

서버 접속

마인크래프트를 실행하고 멀티플레이어에서 공블몬 공식 서버를 클릭하여 접속하세요!
