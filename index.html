서버 소개

공블몬 서버는 포켓몬의 세계관과 크리에이트 모드의 자동화 기능을 결합한 독특한 게임 경험을 제공합니다.

다양한 최적화 모드를 통해 부드러운 게임플레이를 즐기면서, 포켓몬을 수집하고 훈련시키고, 복잡한 자동화 기계를 설계하세요!

Cobblemon과 Create 모드의 조화로운 융합을 통해 창의력을 발휘하고 새로운 차원의 마인크래프트 경험을 누려보세요.

수많은 모드가 완벽하게 최적화되어 있어 원활한 게임 플레이를 보장합니다. 포켓몬 배틀과 자동화 시스템을 동시에 즐기면서 자신만의 스토리를 만들어보세요.

최소 사양

  • CPU: Intel Core i3-6100 / AMD FX-4300 이상

  • RAM: 16GB (8GB 할당 권장)
  • GPU: GTX 1060 / RX 580 이상
  • 저장공간: SSD 10GB 여유 공간
  • 인터넷: 10Mbps 이상 안정적인 연결

자주 묻는 질문

공블몬 서버에 대해 궁금한 점들을 확인해보세요.

서버에 어떻게 접속하나요?
모드팩을 다운로드하고 설치한 후, 마인크래프트에서 멀티플레이어 → 서버 추가 → 서버 주소에 'vp.kro.kr'을 입력하여 접속할 수 있습니다.
모드팩은 어떤 버전을 사용하나요?
현재 마인크래프트 1.20.1 버전 기반의 Fabric 모드로더를 사용합니다. 최신 버전의 모드팩은 v1.2.6입니다.
Java 17이 필요한 이유는 무엇인가요?
마인크래프트 1.20.1과 최신 모드들은 Java 17의 새로운 기능과 성능 개선을 활용하므로 Java 17 이상이 필수입니다.
서버 규칙이 있나요?
기본적인 예의와 매너를 지켜주시면 됩니다. 그리핑, 욕설, 광고는 금지되며, 자세한 규칙은 디스코드 서버에서 확인할 수 있습니다.
모드팩 설치 중 문제가 발생하면 어떻게 하나요?
디스코드 서버의 기술 지원 채널에서 도움을 받을 수 있습니다. 설치 가이드와 문제 해결 방법도 제공됩니다.
